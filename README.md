# GitHub Command Typer

A "MonkeyType" clone for practicing GitHub and Git commands.

## Features

- **Practice Mode**: Type randomized Git commands to improve speed and muscle memory. Tracks WPM and Accuracy.
- **Quiz Mode**: Test your knowledge by typing the correct command for a given action. "Quick Fire" style.
- **Premium Dark Theme**: Developer-centric aesthetic.

## How to Run

1.  `npm install`
2.  `npm run dev`
3.  Open the local URL (usually http://localhost:5173).

## Technologies

- React + Vite
- Vanilla CSS (No large frameworks)
